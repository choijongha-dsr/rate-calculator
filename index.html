감면율 (Reduction of Area)
RA = (1 - d₂² / d₁²) × 100
// d₁ : 입선경, d₂ : 출선경 (mm)

단면적 기준 감소율. 와이어 신선 공정에서 가공도를 나타내는 핵심 지표.

진변형률 (True Strain)
ε = ln(A₀ / A)
= ln(d₁² / d₂²)
// A₀ : 입선 단면적, A : 출선 단면적

누적 진변형률로 인장강도 추정에 활용. 다패스 시 각 패스 값의 합.

단면적
A = (π / 4) × d²
// d : 직경 (mm), A : mm²

원형 단면 기준. π/4 ≈ 0.7854
